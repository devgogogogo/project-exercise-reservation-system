## 🧠 개요
> **전통과 디지털의 만남, 막걸리 플랫폼**  
> 막걸리는 단순한 술이 아닙니다.  
> 지역과 문화, 사람을 연결하는 **매개체이자 이야기의 시작**입니다.

본 프로젝트는 막걸리를 사랑하는 이들을 위한 **통합 플랫폼**으로,  
**검색 · 커뮤니티 · 거래 · 실시간 소통**까지 모든 경험을 한 곳에서 제공합니다.

---

## 🩻 주요 기능

### 💳 온라인 결제
- 막걸리 관련 상품 **쇼핑몰 구매 및 유저 간 중고거래 지원**
- 결제 완료 및 주문 완료 시 **자동 이메일 전송**

### ⚡ 스마트 막걸리 검색
- **페이징 데이터 캐싱** 적용
- 효율적인 데이터 로딩과 빠른 검색 경험 제공

### 📝 막걸리 커뮤니티
- 자유로운 소통이 가능한 **게시판 기능**
- 막걸리 정보 공유, 좋아요, 댓글 작성 가능

### 💬 실시간 채팅
- **WebSocket 기반 1:1 실시간 채팅** 기능 제공
